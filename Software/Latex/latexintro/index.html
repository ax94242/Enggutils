How to set up LaTeX?#

Format thesis, research papers, articles, resume, presentation slides and standard documents with ease using LaTeX. No coding or programming background required. Highly recommended to Academicians. Just see the syntax pattern and get started. I am going to use Windows platform for demonstration.

Pre-Requisites#

Dependency Installation#

  1. Install MiKTEX click here. Click under "Windows" => "Installer" => Click on Download and run the file.
  2. Install Texmaker click here. Download and run the file for Windows.

Great! All dependencies installed.

Getting Started#

Setting up LaTeX#

Next, we want to connect latex library (MiKTEX) with the UI (Texmaker).

  1. Open Texmaker => Click on Options => Configure Texmaker.
  2. Under "Commands" section => Go to "Pdf Viewer" sub-section => Check "Embed".
  3. Under "Quick build" section => select PdfLaTeX + Bib(la)tex + PdfLaTeX(x2) + View Pdf.
  4. Click "OK" button on the bottom.

All set for LaTeX.

Build your first document.#

Copy-paste below code to see your configuration works.

\documentclass{article}
\usepackage{graphicx}
\begin{document}
\title{Introduction to \LaTeX{}}
\author{Author's Name}
\maketitle
\begin{abstract}
The abstract text goes here.
\end{abstract}
\section{Introduction}
Here is the text of your introduction.
\begin{equation}
\label{simple_equation}
\alpha = \sqrt{ \beta }
\end{equation}
\subsection{Subsection Heading Here}
Write your subsection text here.
\section{Conclusion}
Write your conclusion here.
\end{document}

Click on Arrow icon left to "Quick Build" and display your formatted pdf document.

I personally use a onestep guide from overleaf https://www.overleaf.com/learn/latex/Tutorials for finding required LaTeX syntax.
