How to set up BoltzTrap?#

Install BoltzTrap to calculate electronic and thermal properties based on the K-path points and the band gap solution for a material.

Pre-Requisites#

Dependency Installation#

  1. Download Boltztrap click here. Click under "Windows" => "Installer" => Click on Download and run the file.
  2. Install Virtual Box click here. Download and run the file for Windows.
  3. Download Linux Image click here.

Getting Started#

Setting up LaTeX#

Open Virtual Box software and load the downloaded Linux Image. Once loaded open the command line in the Linux system and execute below command. Follow the tutorial to execute below configuration properly : http://gestyy.com/eot8dy.

Build your first document.#

Copy-paste below code to see your configuration works.

1) tar xvf $HOME/Downloads/BoltzTraP.tar.bz2
2) cd boltz*
3) cd src
4) gedit Makefile
---------
FC=ifort
FOPT = -FR -mp1 -w -prec_div -pc80 -pad -align -DINTEL_VML -traceback -O2
LDFLAGS = $(FOPT) -L$(MKLROOT)/lib/$(MKL_TARGET_ARCH) -pthread
LIBS = -lmkl_lapack95_lp64 -lmkl_intel_lp64 -lmkl_intel_thread -lmkl_core -openmp -lpthread
----------
5) make clean
6) make
