Create pages with reactjs#

Below is the format of the reactjs for creating pages. If you want to create simple page with little bit of html code experience, below is the basic syntax format. Enter your code in between <div> </div>

import React from 'react';
function Hello() {
return (
<div>
<p>
<code>Your code goes here!</code>
</p>
</div>
);
}
export default Hello;
