  1. How to add image in my Webpages? Webpaages are created using Markdown.

    ![Locale Dropdown](/img/tutorial/localeDropdown.png)

  2. Markdown Quick sheet Wordpress has a good quick sheet to refer formatting: https://wordpress.com/support/markdown-quick-reference/

  3. How to Embed a Video or pdf or any document Easy approach would be to host document in Google drive, PCloud or any other drive. Replace link in below code:

    import React from 'react';
    import Layout from '@theme/Layout';
    export const Hello = () => {
    return(
    <Layout title="Hello">
    <div className='container'>
    <h1>embeded file</h1>
    <iframe src="https://www.sfda.gov.sa/sites/default/files/2019-10/Payment-disclosure-en.pdf" width="1040" height="480"></iframe>
    </div>
    </Layout>
    )
    }
    export default Hello
