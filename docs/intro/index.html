How to build your website with Docusaurus?

Easy to make website without any coding or programming background. Highly recommended to Academicians. Just see the syntax pattern and get started with Docusaurus in less than 5 minutes. I am going demonstrate using Windows Platform. For other operating version, you may follow new.docusaurus.io.

Pre-Requisites#

We are going to use Git, Node.js, yarn and github for hosting website. This may look complicated but trust me, in the end it will worth it.

Dependency Installation#

  1. Install Git click here. Follow the instruction for windows operating system.
  2. Install Node.js click here. Follow the instruction for windows operating system.
  3. Install Yarn. This was an optional before but with the new release of docusaurus, Yarn became necessity. Simply search "Git" in your windows search bar and open it. Run below command to install it:
npm install --global yarn

Congrats! You made this far. It was not that complicated afterall.

At last, we will set up Github Page for your Website: 4. Go to Github and sign up for an account. a. Go to your profile section, can be found at the right upper corner. Next to the profile picture on right corner, Click on plus icon and select "New repository". b. You may have to create public repository for the free acount.

Great! All dependencies installed.

Getting Started#

Time to set up website locally in your computer. So, that you can work offline and deploy the website, when it's finalized.

  1. Open your "Git" again.
  2. Change your directory where you want to save the website files locally. For an instance, Run command something like below:
cd ~/docs/github/enggutils
  1. Clone the github repository to your local computer. Change USERNAME with your Github user name
git clone https://github.com/USERNAME/docusaurus-tutorial.git

Get started by creating a new site.

Or try Docusaurus immediately with new.docusaurus.io.

Generate a new site#

Generate a new Docusaurus site using the classic template:

npx @docusaurus/init@latest init my-website classic

Start your site#

Run the development server:

cd my-website
npx docusaurus start

Your site starts at http://localhost:3000.

Open docs/intro.md and edit some lines: the site reloads automatically and display your changes.
